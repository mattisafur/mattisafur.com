Create The virtual machines talos will be installed on, for a single node proxmox cluster, 1 control plane and 1 worker node should be sufficient.

If you have multiple nodes in your proxmox cluster, adding a control plane on each proxmox node will be good to allow for high availablity. In addition, adding more worker nodes will be needed to achieve high availability.
